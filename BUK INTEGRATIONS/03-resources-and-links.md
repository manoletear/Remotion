- **Social Media** - Animated posts, reels, and stories
- **Data visualization** - Charts and graphs that animate
- **Marketing** - Product demos and promotional videos

## AI Integration

### Prompting Videos with Claude Code

See: https://www.remotion.dev/docs/ai/claude-code

Remotion supports AI-assisted video creation via Claude Code. The `npx create-video@latest` command can be used to scaffold projects that work well with AI prompting.
